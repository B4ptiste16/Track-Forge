AC TRACK FORGE — PROJECT HANDOFF (read this first in a new chat)
================================================================
What it is: a desktop app to DESIGN Assetto Corsa circuits and export them.
You draw a centerline/segments; it builds the track mesh, kerbs, trackside
strips, pit lane, buildings, and the AI racing line (fast_lane.ai), then exports
an FBX + config so the track loads in AC (and its ai line feeds the AC-RL bot).

Stack: Electron (electron-builder NSIS, Windows ARM64) + electron-updater +
React + TypeScript + Vite + three.js (Z-up) + jszip. Custom ASCII FBX 7.4 writer.

BUILD / RELEASE:
- Version bumps: use a bump SCRIPT FILE (bump*.js), NOT `node -e "..."` — PowerShell

ARCHITECTURE — the geometry engine is the single source of truth:
- src/types.ts        all data shapes. Kerbs have per-part widths
                      (entryW/apexW/exitW) + unclamped apexLen. Trackside =
                      {left,right,zones} of StripCfg{texture,width,wall,wallDist}
                      with StripTexture grass|gravel|gravel_spaced|concrete.
                      Building{}, SavedLayout, pit.structures.
- src/state/project.ts  defaults, migration (legacy runoff -> zones), layout
                      snapshot/apply, color map.
- src/geometry/       kerbs.ts (tapered triangle ends, per-part widths),
                      runoff.ts (continuous trackside strip; smoothCeiling with
                      erosion-guarded blur to kill corner "wedge" artifacts),
                      build.ts (assembles mesh, mergeInto), pitlane.ts,
                      buildings.ts.
- src/export/ai.ts    genFastLaneAi() — rotates spline so point 0 = start/finish
                      line (fixes phantom ~3s laps + bot progress origin).
- src/components/     App.tsx (tabbed right panel: Track/Kerbs/Trackside/
                      Pit & Buildings + layout save/switch toolbar),
                      TracksidePanel, FacilitiesPanel, SegmentEditor2D
                      (modes: segment/zone/building). RunoffPanel was DELETED.
